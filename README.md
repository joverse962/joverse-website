## üî≠ About The Project

**JOVERSE** serves as the central hub and documentation portfolio for a series of high-performance engineering projects. It is built to showcase work at the intersection of **embedded systems**, **computer vision**, and **autonomous robotics**.

The website itself is a modern, high-performance static site built with **Astro** and **React**, designed for speed and SEO optimization while delivering a rich, interactive user experience.

## üíª Web Architecture

This portfolio is engineered for performance using a modern "Islands Architecture" approach.

### ‚ö° Tech Stack
* **Core Framework:** [Astro](https://astro.build/) (Static Site Generation)
* **UI Components:** [React](https://react.dev/) (Interactive Elements)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Deployment:** [Netlify](https://www.netlify.com/)

### üé® Key Features
* **Sticky "Island" Header:** A custom React component that transitions from a floating island to a full-width bar on scroll.
* **Glassmorphism UI:** Heavy use of backdrop filters and semi-transparent borders (colors: *Woodsmoke*, *Cool Sky*, *Crail*).
* **Responsive Design:** Fully fluid layouts that adapt from mobile to ultra-wide desktop screens.
